# AgentScope

AgentScope 是阿里巴巴 ModelScope 团队开发的多智能体框架，支持 Python 和 Java 两种语言实现。

## 概述

AgentScope 是一个用于构建多智能体应用的开源框架，提供了完整的智能体开发、运行和管理能力。

## 核心组件

### AgentScope (Python)

Python 版本的核心框架，提供：
- 智能体定义与管理
- 多智能体协作机制
- 消息传递系统
- 工具调用接口

### AgentScope-Java

Java 版本的实现，适用于企业级 Java 生态。

### AgentScope-Runtime

运行时环境，支持：
- 智能体执行
- 云原生扩展
- 计算和存储资源管理

### AgentScope-Studio

可视化开发和管理界面。

### ReMe

AgentScope 的内存系统别名，提供文件型记忆管理能力。
