# QwenTeam

## 简介

QwenTeam 是阿里巴巴达摩院通义实验室的大模型研发团队，负责 Qwen（通义千问）系列大语言模型的开发与维护。

## 主要产品

- [[Qwen]] 系列大语言模型
- Qwen Code - 终端设计的编码智能体
- Qwen VL - 视觉语言模型
