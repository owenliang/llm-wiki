# Skills Hub

Skills Hub 是 AI Agent 平台的技能/插件市场，用户可以从中下载和安装扩展功能。

## 概念

Skill（技能）是 Agent 的能力扩展单元，通常包含：
- 工具定义
- 调用逻辑
- 配置参数

## 主要 Skills Hub

| Hub | 支持平台 | 网址 |
|-----|----------|------|
| ClawHub | [[OpenClaw]]、[[CoPaw]] | https://clawhub.ai/ |
| Skills.sh | [[CoPaw]] | https://skills.sh/ |
| SkillsMP | [[CoPaw]] | https://skillsmp.com/ |
| GitHub | [[CoPaw]] | https://github.com/ |

- 代码审查
- 沙箱执行
